# coffeeStore
커피 스토어 (2021 dev-matching - 커피 주문 페이지 만들기 SPA)

vsCode 에서 터미널을 두개 띄워놓고 각각의 서버를 켜준다.

1. json 서버 켜기
npx json-server ./data.json --port 4000
임시로 API 에서 데이터를 받아오는 것 처럼 할 수 있다.

2. Web 서버 켜기
npx serve -s
